工作流快照

1. 内容确定

维护 content/knowledge.md，先保证信息结构和准确性。

2. 设计与改版

维护 src/template.html 和 src/styles.css。

3. 构建与检查

执行 npm run build 与 npm run preview 复核。

4. 自动上线

推送到 GitHub 后，Vercel 会自动触发重新部署。
