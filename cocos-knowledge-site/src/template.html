工作流快照

1. 内容确定

维护 content/knowledge.md，优先信息结构和准确性。

2. 设计迭代

维护 src/template.html 与 src/styles.css。

3. 构建检查

执行 npm run build 与 npm run preview 复核。

4. 自动上线

推送到 GitHub 后，Vercel 自动触发重新部署。
